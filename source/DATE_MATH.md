# DATE MATH

- `DATEDIFF(date1, date2)` : 두 날짜 간의 일수 차이를 계산하여 일(day) 단위로 반환

      ```sql
      mysql> SELECT name,
          ->        birthDATE,
          ->        DATEDIFF(NOW(), birthDATE) AS "Days after birth"
          -> FROM birth;
      +--------+------------+------------------+
      | name   | birthDATE  | Days after birth |
      +--------+------------+------------------+
      | Thomas | 2020-02-02 |             1167 |
      | Ron    | 2021-07-12 |              641 |
      | Woody  | 2022-10-10 |              186 |
      | Drew   | 2023-04-14 |                0 |
      +--------+------------+------------------+
      4 rows in set (0.00 sec)
      ```

  <br>

- `DATE` ADD

  ```sql
  -- USE DATE_ADD function

  mysql> SELECT birthDT,
      ->        DATE_ADD(birthDT, INTERVAL 1 MONTH) AS "One month later"
      -> FROM birth;
  +---------------------+---------------------+
  | birthDT             | One month later     |
  +---------------------+---------------------+
  | 2020-02-02 10:00:01 | 2020-03-02 10:00:01 |
  | 2021-07-12 08:08:08 | 2021-08-12 08:08:08 |
  | 2022-10-10 22:23:24 | 2022-11-10 22:23:24 |
  | 2023-04-14 21:46:05 | 2023-05-14 21:46:05 |
  +---------------------+---------------------+
  4 rows in set (0.00 sec)

  -- USE + operator

  mysql> SELECT birthDT,
      ->        birthDT + INTERVAL 1 MONTH AS "One month later"
      -> FROM birth;
  +---------------------+---------------------+
  | birthDT             | One month later     |
  +---------------------+---------------------+
  | 2020-02-02 10:00:01 | 2020-03-02 10:00:01 |
  | 2021-07-12 08:08:08 | 2021-08-12 08:08:08 |
  | 2022-10-10 22:23:24 | 2022-11-10 22:23:24 |
  | 2023-04-14 21:46:05 | 2023-05-14 21:46:05 |
  +---------------------+---------------------+
  4 rows in set (0.00 sec)
  ```

  ```sql
  mysql> SELECT birthDT,
      ->        birthDT + INTERVAL 10 MONTH + INTERVAL 10 MINUTE
      ->        AS "10 months, 10 minutes later"
      -> FROM birth;
  +---------------------+-----------------------------+
  | birthDT             | 10 months, 10 minutes later |
  +---------------------+-----------------------------+
  | 2020-02-02 10:00:01 | 2020-12-02 10:10:01         |
  | 2021-07-12 08:08:08 | 2022-05-12 08:18:08         |
  | 2022-10-10 22:23:24 | 2023-08-10 22:33:24         |
  | 2023-04-14 21:46:05 | 2024-02-14 21:56:05         |
  +---------------------+-----------------------------+
  4 rows in set (0.00 sec)
  ```

- `DATE` SUB

  ```sql
  -- USE DATE_SUB function

  mysql> SELECT birthDT,
      ->        DATE_SUB(birthDT, INTERVAL 1 MONTH) AS "A month ago"
      -> FROM birth;
  +---------------------+---------------------+
  | birthDT             | A month ago         |
  +---------------------+---------------------+
  | 2020-02-02 10:00:01 | 2020-01-02 10:00:01 |
  | 2021-07-12 08:08:08 | 2021-06-12 08:08:08 |
  | 2022-10-10 22:23:24 | 2022-09-10 22:23:24 |
  | 2023-04-14 21:46:05 | 2023-03-14 21:46:05 |
  +---------------------+---------------------+
  4 rows in set (0.00 sec)

  -- USE - operator

  mysql> SELECT birthDT,
      ->        birthDT - INTERVAL 1 MONTH AS "A month ago"
  ```
