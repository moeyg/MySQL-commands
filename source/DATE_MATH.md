# DATE MATH

- `DATEDIFF(date1, date2)` : 두 날짜 간의 일수 차이를 계산하여 일(day) 단위로 반환

   ```sql
   SELECT NAME,
          MFG_DATE,
          DATEDIFF(NOW(), MFG_DATE) AS "- DAY PRODUCED"
   FROM PRODUCT_INFO;
   +-------------+------------+----------------+
   | NAME        | MFG_DATE   | - DAY PRODUCED |
   +-------------+------------+----------------+
   | CocaCola    | 2023-03-03 |             50 |
   | Sprite      | 2023-04-24 |             -2 |
   | PocariSweat | 2023-01-01 |            111 |
   | Pepsi       | 2023-06-06 |            -45 |
   | Monster     | 2023-04-21 |              1 |
   +-------------+------------+----------------+
   ```

  <br>

- `DATE` ADD

  ```sql
  -- USE DATE_ADD function

  SELECT EXP,
         DATE_ADD(EXP, INTERVAL 1 MONTH) AS "ONE MONTH LATER"
  FROM PRODUCT_INFO;
  +---------------------+---------------------+
  | EXP                 | ONE MONTH LATER     |
  +---------------------+---------------------+
  | 2025-05-05 05:05:05 | 2025-06-05 05:05:05 |
  | 2025-01-31 23:00:10 | 2025-02-28 23:00:10 |
  | 2024-12-28 01:20:28 | 2025-01-28 01:20:28 |
  | 2026-11-28 09:09:09 | 2026-12-28 09:09:09 |
  | 2023-04-21 15:11:04 | 2023-05-21 15:11:04 |
  +---------------------+---------------------+
  ```
  ```sql
  -- USE + operator

  SELECT EXP,
         EXP + INTERVAL 1 MONTH AS "ONE MONTH LATER"
  FROM PRODUCT_INFO;
  +---------------------+---------------------+
  | EXP                 | ONE MONTH LATER     |
  +---------------------+---------------------+
  | 2025-05-05 05:05:05 | 2025-06-05 05:05:05 |
  | 2025-01-31 23:00:10 | 2025-02-28 23:00:10 |
  | 2024-12-28 01:20:28 | 2025-01-28 01:20:28 |
  | 2026-11-28 09:09:09 | 2026-12-28 09:09:09 |
  | 2023-04-21 15:11:04 | 2023-05-21 15:11:04 |
  +---------------------+---------------------+
  ```

  ```sql
  SELECT EXP,
         EXP + INTERVAL 10 MONTH + INTERVAL 10 MINUTE
         AS "10 MONTHS, 10 MINUTES LATER"
  FROM PRODUCT_INFO;
  +---------------------+-----------------------------+
  | EXP                 | 10 MONTHS, 10 MINUTES LATER |
  +---------------------+-----------------------------+
  | 2025-05-05 05:05:05 | 2026-03-05 05:15:05         |
  | 2025-01-31 23:00:10 | 2025-11-30 23:10:10         |
  | 2024-12-28 01:20:28 | 2025-10-28 01:30:28         |
  | 2026-11-28 09:09:09 | 2027-09-28 09:19:09         |
  | 2023-04-21 15:11:04 | 2024-02-21 15:21:04         |
  +---------------------+-----------------------------+
  ```

- `DATE` SUB

  ```sql
  -- USE DATE_SUB function

  SELECT EXP,
         DATE_SUB(EXP, INTERVAL 1 MONTH) AS "A MONTH AGO"
  FROM PRODUCT_INFO;
  +---------------------+---------------------+
  | EXP                 | A MONTH AGO         |
  +---------------------+---------------------+
  | 2025-05-05 05:05:05 | 2025-04-05 05:05:05 |
  | 2025-01-31 23:00:10 | 2024-12-31 23:00:10 |
  | 2024-12-28 01:20:28 | 2024-11-28 01:20:28 |
  | 2026-11-28 09:09:09 | 2026-10-28 09:09:09 |
  | 2023-04-21 15:11:04 | 2023-03-21 15:11:04 |
  +---------------------+---------------------+
  ```
  ```sql
  -- USE - operator

  SELECT EXP,
         EXP - INTERVAL 1 MONTH AS "A MONTH AGO"
  FROM PRODUCT_INFO;
  +---------------------+---------------------+
  | EXP                 | A MONTH AGO         |
  +---------------------+---------------------+
  | 2025-05-05 05:05:05 | 2025-04-05 05:05:05 |
  | 2025-01-31 23:00:10 | 2024-12-31 23:00:10 |
  | 2024-12-28 01:20:28 | 2024-11-28 01:20:28 |
  | 2026-11-28 09:09:09 | 2026-10-28 09:09:09 |
  | 2023-04-21 15:11:04 | 2023-03-21 15:11:04 |
  +---------------------+---------------------+
  ```
  ```sql
  SELECT EXP,
         EXP - INTERVAL 10 MONTH - INTERVAL 10 HOUR
         AS "10 MONTHS 10 MONTHS, 10 HOURS AGO"
  FROM PRODUCT_INFO;
  +---------------------+-----------------------------------+ 
  | EXP                 | 10 MONTHS 10 MONTHS, 10 HOURS AGO |
  +---------------------+-----------------------------------+
  | 2025-05-05 05:05:05 | 2024-07-04 19:05:05               |
  | 2025-01-31 23:00:10 | 2024-03-31 13:00:10               |
  | 2024-12-28 01:20:28 | 2024-02-27 15:20:28               |
  | 2026-11-28 09:09:09 | 2026-01-27 23:09:09               |
  | 2023-04-21 15:11:04 | 2022-06-21 05:11:04               |
  +---------------------+-----------------------------------+
  ```
